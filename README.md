# Rekap Crew Dentanet

Repo ini dipakai Alvii untuk menyimpan rekap jadwal maintenance/perbaikan crew Dentanet dari WhatsApp group yang diizinkan oleh Tuan Besar.

## Aturan utama
- Alvii tidak membalas pesan di group sumber.
- Rekap harian dijalankan sekitar jam 15:00 WITA.
- Semua jadwal/list maintenance yang dikirim teknisi dicatat.
- Centang hijau tidak diabaikan; jika terlihat, dicatat sebagai marker/status saja.
- Chat yang tidak terkait jadwal maintenance tidak dimasukkan.

## Struktur
- `blueprint/` — SOP dan aturan kerja.
- `rekap/YYYY-MM-DD.md` — hasil rekap harian.
- `raw/screenshots/` — referensi screenshot bila diperlukan.
